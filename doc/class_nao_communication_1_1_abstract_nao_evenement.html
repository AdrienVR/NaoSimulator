def NaoCommunication.AbstractNaoEvenement.arreterReconnaissanceVocale (   self)

Arrête la reconnaissance vocale.

arreterReconnaissanceVocale Cette fonction arrête la reconnaissance vocale.
Pour stopper la reconnaissance vocale nous devons se désinscrire de l'évènement de reconnaissance naoEvenement.arreterReconnaissanceVocale()
Voir les commentaires de l'exemple pour en comprendre le fonctionnement.

Précondition
nao.attribuerVocabulaire
naoEvenement.demarrerReconnaissanceVocale()
Postcondition
naoEvenement.arreterReconnaissanceVocale()

Exemple :

1 nao.attribuerVocabulaire("bonjour","pomme") #on attribue des nouveaux mots de vocabulaire à Nao
2 naoEvenement.demarrerReconnaissanceVocale() #Inscription à l'évènement de reconnaissance vocale
3 while not naoEvenement.recoVocale: #Tant que Nao n'a pas entendu de mot (naoEvenement.recoVocale = faux) la reconnaissance vocale est en cours
4  time.sleep(0.5) #pause de 0.5 sec
5 if naoEvenement.recoVocale: #dès que Nao a reconnu un mot (naoEvenement.recoVocale = vrai)
6  nao.dire("J'ai reconnu le mot %s" %(naoEvenement.motReconnu)) #Nao dis le mot qu'il a reconnu (le mot est stocké dans naoEvenement.motReconnu)
7  naoEvenement.arreterReconnaissanceVocale() # on se désinscrit de l'évènement de reconnaissance vocale
